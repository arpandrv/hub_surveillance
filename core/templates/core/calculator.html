Sample Size Formula

The calculation uses the formula for proportion estimation with finite population correction:

n = (N × z² × p(1-p)) ÷ (d²(N-1) + z²p(1-p))

Where:
- N = {{ calculation_results.N }} (total plants)
- z = {{ calculation_results.z }} (z-score for {{ calculation_results.confidence_level_percent }}% confidence)
- p = {{ calculation_results.p }} ({{ calculation_results.p_percent }}% expected prevalence in {{ calculation_results.season }} season)
- d = {{ calculation_results.d }} (5% margin of error)
